# $interactionUpdateDefer

## Описание
Создаёт отсрочку для интерактивного обновления, на случай если обновление будет более чем через 3 секунды.

## Использование
```js
$interactionUpdateDefer
```

### Опции
Отсутствуют

## Пример использования
```js
$interactionUpdate[На кнопку нажал <@$authorID>]
$interactionFollowUp
$interactionUpdateDefer
```
