# $title

## Описание
Добавляет заголовок к сообщению. Эмбед функция.
## Использование
```js
$title[индекс;текст;ссылка]
```

### Опции
- `Индекс` - индекс сообщения
- `Teкст` - текст поля 
- `Ссылка` - кликабельная ссылка на тексте заголовка

## Пример использования
```js
$title[1;$userTag[$clientID];$getBotInvite]
```
