# $interactionDefer

## Описание
Создаёт отсрочку для интерактивного ответа, на случай если ответ будет более чем через 3 секунды.

## Использование
```js
$interactionDefer
```

### Опции
Отсутствуют

## Пример использования
```js
$interactionReply[Сейчас играет - $songInfo[title]]
$interactionFollowUp
$playTrack[youtube;$slashOption[track]]
$interactionDefer
```
