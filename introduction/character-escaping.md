---
Описание: >-
  С помощью этих символов вы можете задавать боту символы не принимая их как конец функции или разделитель
---

# Заменители символов

`[` => `#RIGHT#`

`]` => `#LEFT#`

`;` => `#SEMI#`

`:` => `#COLON#`

`$` => `#CHAR#`

`>` => `#RIGHT_CLICK#`

`<` => `#LEFT_CLICK#`

`=` => `#EQUAL#`

`}` => `#LEFT_BRACKET#`

`{` => `#RIGHT_BRACKET#`

## Больше:

Назад => `\b`\
Новая форма => `\f`\
Новая строка => `\n`\
Горизонтальный табулятор => \
Вертикальный табулятор => `\v`\
Одинарная кавычка => `\'`\
Двойная кавычка => `\"`\
